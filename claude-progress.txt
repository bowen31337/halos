## Technical Notes

### Dependencies to Install
```bash
# Backend (Python)
uv pip install fastapi uvicorn sqlalchemy pydantic sse-starlette langchain-anthropic langgraph

# Frontend (Node.js)
pnpm create vite@latest client -- --template react-ts
pnpm add zustand react-router-dom @tanstack/react-query tailwindcss
```

### Running the Application
```bash
# Backend
source .venv/bin/activate
uvicorn src.main:app --reload --port 8000

# Frontend (once set up)
cd client && pnpm dev
```

### API Documentation
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
- OpenAPI JSON: http://localhost:8000/openapi.json
